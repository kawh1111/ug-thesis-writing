# 中文学术写作风格指南

## 一、基本原则

- 客观、克制、以证据为基础
- 短句优先，清晰表达
- 区分事实、结果和解释

## 二、推荐句式

### 结果描述
- "表X-Y列出了..."
- "从图X-Y可以看出..."
- "结果表明..."
- "进一步分析发现..."
- "对比各模型表现，..."

### 对比分析
- "与...相比，..."
- "...优于/劣于..."
- "在...条件下，...表现更为突出"
- "值得注意的是，..."

### 局限性说明
- "本研究存在以下局限：..."
- "由于...的限制，..."
- "未来研究可从以下方面改进：..."

### 数据描述
- "研究区位于..."
- "数据来源为..."
- "时间范围为..."
- "共包含...个样本"

## 三、禁止用语

### 夸张类
- ~~"首次提出"~~ → "本文提出"
- ~~"颠覆性"~~ → "具有一定的改进"
- ~~"降维打击"~~ → "显著优于"
- ~~"吊打"~~ → "表现更优"
- ~~"黑科技"~~ → "创新方法"

### AI 特征类（AIGC 检测高风险）
- ~~"值得注意的是"~~ → 直接陈述
- ~~"总而言之"~~ → 删除或用"综上"
- ~~"不难发现"~~ → "可以看出"
- ~~"毋庸置疑"~~ → 删除（有引用支撑则保留）
- ~~"众所周知"~~ → 删除或加引用

### 博客/产品体
- ~~"让我们来看看"~~ → 删除
- ~~"接下来我将"~~ → 删除
- ~~"首先...其次...最后..."~~（三段式）→ 变化句式

## 四、术语标准化

全文必须使用同一套术语，以下为常见混用示例：

| 推荐 | 避免 |
|------|------|
| 研究区 | 研究区域、研究范围（混用） |
| 测试集 | 测试数据、测试样本（混用） |
| 训练集 | 训练数据、训练样本（混用） |
| 评价指标 | 评估指标、衡量指标（混用） |
| 样本量 | 样本数量、样本数目（混用） |
| 预测精度 | 预测准确度、预测准确率（混用） |

## 五、段落结构

### 标准段落（3-5句）
1. 主题句：点明本段核心
2. 展开句：提供证据或分析
3. 过渡句：连接下一段或总结本段

### 图表分析段落
1. 说明内容："图X-Y展示了..."
2. 解释模式："从图中可以看出..."
3. 讨论含义："这一结果表明..."

## 六、引用规范

- 每段至少1个引用（绪论和文献综述部分）
- 引用格式与学校要求一致
- 禁止无引用断言（"研究表明"必须有具体引用）
- 引用位置：句末或作者名后

## 七、数值与单位

- 数值与单位之间加空格：12 pt、1.5 倍
- 百分比：30%（无空格）
- 范围：2018—2023（用一字线）
- 小数：保留适当位数（通常2-3位）
