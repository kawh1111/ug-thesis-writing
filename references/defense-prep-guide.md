# 答辩准备指南

## 一、PPT 制作

### 结构模板（20-25页）

| 页码 | 内容 | 时长 |
|------|------|------|
| 1 | 封面（标题、姓名、导师、学校） | 30秒 |
| 2 | 目录 | 20秒 |
| 3-4 | 研究区与数据概况 | 1分钟 |
| 5-6 | 研究背景与问题 | 1分钟 |
| 7 | 研究目标与创新点 | 30秒 |
| 8 | 技术路线 | 30秒 |
| 9-10 | 数据基础与预测任务 | 1分钟 |
| 11-13 | 模型对比结果 | 1.5分钟 |
| 14-15 | 模型可解释性分析 | 1分钟 |
| 16-18 | 情景预测结果 | 1.5分钟 |
| 19 | 结论 | 30秒 |
| 20 | 展望与不足 | 20秒 |
| 21 | 致谢 | 10秒 |

**总时长**: 8-10分钟

### 设计规范

- 比例: 16:9
- 字体: 标题黑体/微软雅黑，正文宋体/微软雅黑
- 字号: 标题≥28pt，正文≥20pt
- 配色: 使用学校模板配色
- 图表: 直接从论文中选取关键图表

### PPT 内容要点

#### 封面页
- 论文标题（完整）
- 答辩人姓名、学号
- 指导教师
- 学校、学院、专业
- 日期

#### 目录页
- 清晰列出各部分标题
- 使用编号或图标

#### 研究区页
- 研究区位置图
- 关键地理特征
- 数据覆盖范围

#### 技术路线页
- 流程图形式
- 数据层→建模层→解释层
- 突出创新点

#### 结果页
- 使用论文中的核心图表
- 突出关键数据（R²、RMSE等）
- 对比图优于表格

#### 结论页
- 3-5条核心结论
- 每条对应一个研究目标
- 简洁明了

## 二、讲稿撰写

### 基本原则

- 口语化，不要念PPT
- 每页讲稿控制在30-60秒
- 关键数据要记住
- 准备3分钟压缩版（备用）

### 讲稿结构

```
各位老师好，我是[姓名]。我的毕业论文题目是《[标题]》，指导教师是[导师]。

[开场白：一句话概括研究内容]

下面按目录顺序向各位老师汇报。

[逐页讲稿...]

以上是我的汇报，请各位老师批评指正。
```

### 关键口径

- 不要讲成"精确预测"，要讲成"建立流程"
- 强调"数据有限条件下的可行方案"
- 突出"统一比较框架"而非"最优模型"
- 承认局限性，展示学术态度

### 3分钟压缩版

```
各位老师好，我是[姓名]。我的论文研究[一句话概括]。

核心工作有三点：
第一，[数据工作]
第二，[模型比较]
第三，[解释分析]

主要结论是：[2-3句]

存在以下局限：[1-2句]

以上是我的汇报，请各位老师批评指正。
```

## 三、Q&A 准备

### 常见问题分类

#### 方法类（10个）
1. 为什么选择这些模型？
2. 为什么用这几个评价指标？
3. 训练集/验证集/测试集怎么划分的？
4. 为什么不做随机K折交叉验证？
5. 超参数怎么调的？
6. 特征工程具体做了什么？
7. 为什么用这种可解释性方法而不是其他方法？
8. 滞后特征的窗口大小是怎么确定的？
9. 样本标识编码的目的是什么？
10. 模型训练用了多长时间？

#### 数据类（5个）
1. 数据来源是什么？
2. 缺失值怎么处理的？
3. 为什么选择这个时间段的数据？
4. 样本是怎么选取的？
5. 多源数据是怎么对齐的？

#### 结果类（5个）
1. 为什么某个模型在某些任务上表现更好？
2. 可解释性分析的主要发现是什么？
3. 预测方案的假设依据是什么？
4. 不同样本的预测精度差异大吗？
5. 模型的泛化能力如何？

#### 局限性类（5个）
1. 研究的主要局限是什么？
2. 数据量是否足够？
3. 缺少某类数据的影响？
4. 模型是否过拟合？
5. 结论的适用范围是什么？

### 回答模板

```
[直接回答问题]
[补充1-2个支撑点]
[如果不确定，诚实说明]
```

**示例**:
> Q: 为什么选择这个模型？
> A: 选择它有三个考虑：一是它在同类研究中有较好的基准表现；二是它训练速度快，便于调参；三是它支持可解释性分析，便于理解预测依据。
