# 格式检查清单

## 一、字号对照表

| 中文字号 | 磅值(pt) | 常见用途 |
|----------|----------|----------|
| 二号 | 22pt | 论文标题 |
| 三号 | 16pt | 章标题 |
| 四号 | 14pt | 节标题 / 摘要标题 |
| 小四号 | 12pt | 正文 |
| 五号 | 10.5pt | 图表名 / 脚注 |
| 小五号 | 9pt | 页眉页脚 |

## 二、格式检查项

### A. 正文格式

| 检查项 | 标准值 | 常见错误 |
|--------|--------|----------|
| 字体 | 宋体 | 混用其他字体 |
| 字号 | 小四号(12pt) | 9pt或10.5pt |
| 行距 | 1.5倍 | 1.0/1.2倍 |
| 首行缩进 | 2字符 | 无缩进 |
| 对齐 | 两端对齐 | 未设置 |
| 西文/数字 | Times New Roman | 混用Arial等 |

### B. 标题格式

| 层级 | 字体 | 字号 | 加粗 | 对齐 |
|------|------|------|------|------|
| 章标题(第X章) | 黑体 | 三号(16pt) | 是 | 居中 |
| 节标题(X.X) | 黑体/宋体 | 四号/小四号 | 是 | 左对齐 |
| 条标题(X.X.X) | 宋体 | 小四号 | 视学校要求 | 左对齐 |

**注意**: 学院要求与学校模板可能冲突，需确认以哪个为准。

### C. 摘要格式

| 项目 | 字体 | 字号 |
|------|------|------|
| "摘要"标题 | 黑体 | 四号(14pt) |
| 摘要正文 | 宋体 | 小四号(12pt) |
| "关键词"标签 | 宋体加粗 | 小四号(12pt) |
| "ABSTRACT"标题 | Times New Roman加粗 | 四号(14pt) |
| 英文摘要正文 | Times New Roman | 小四号(12pt) |
| "Keywords"标签 | Times New Roman加粗 | 小四号(12pt) |

### D. 目录格式

| 项目 | 字体 | 字号 |
|------|------|------|
| "目录"标题 | 黑体 | 三号(16pt) |
| 章条目 | 宋体加粗 | 小四号(12pt) |
| 节条目 | 宋体 | 小四号(12pt) |
| 行距 | 1.5倍 | - |

### E. 参考文献格式

| 项目 | 标准 |
|------|------|
| 标题 | 黑体四号(14pt) |
| 条目字号 | 宋体小四号(12pt) |
| 行距 | 1.5倍 |
| 格式 | GB/T 7714 或学校要求 |

### F. 图表格式

| 项目 | 标准 |
|------|------|
| 图名位置 | 图下方居中 |
| 表名位置 | 表上方居中 |
| 字体 | 宋体五号(10.5pt) |
| 编号 | 图X-Y / 表X-Y |
| 分辨率 | ≥300 DPI |

### G. 页面设置

| 项目 | 标准 |
|------|------|
| 纸张 | A4 |
| 上边距 | 2.54 cm (1 in) |
| 下边距 | 2.54 cm (1 in) |
| 左边距 | 3.17 cm (1.25 in) |
| 右边距 | 3.17 cm (1.25 in) |
| 装订线 | 左侧 |

## 三、严重程度分级

### 严重（必须修正）
- 正文字号错误
- 行距不一致
- 标题层级格式混乱

### 中等（建议修正）
- 首行缩进缺失
- 对齐方式不一致
- 图表名字号错误

### 轻微（可选修正）
- 页边距微小差异
- 页眉页脚格式
- 空白页处理

## 四、格式配置文件模板

创建 `format.json` 用于脚本检查：

```json
{
  "body": {
    "font_cn": "宋体",
    "font_en": "Times New Roman",
    "size_pt": 12,
    "line_spacing": 1.5,
    "first_indent_chars": 2,
    "alignment": "both"
  },
  "heading1": {
    "font_cn": "黑体",
    "size_pt": 16,
    "bold": true,
    "alignment": "center"
  },
  "heading2": {
    "font_cn": "黑体",
    "size_pt": 14,
    "bold": true,
    "alignment": "left"
  },
  "heading3": {
    "font_cn": "宋体",
    "size_pt": 12,
    "bold": false,
    "alignment": "left"
  },
  "caption": {
    "font_cn": "宋体",
    "size_pt": 10.5,
    "alignment": "center"
  },
  "page": {
    "paper": "A4",
    "margin_top_cm": 2.54,
    "margin_bottom_cm": 2.54,
    "margin_left_cm": 3.17,
    "margin_right_cm": 3.17
  }
}
```
